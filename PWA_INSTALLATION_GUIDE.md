# NRCS EAM Progressive Web App (PWA) Installation Guide

The NRCS Enterprise Asset Management system is a Progressive Web App (PWA) that can be installed on various devices for quick access and offline functionality.

## What is a PWA?

A Progressive Web App combines the best features of web and mobile apps:
- **Installable**: Can be installed on your device like a native app
- **Offline Access**: Works even without internet connection (read-only mode)
- **Fast Loading**: Cached resources load instantly
- **No App Store Required**: Install directly from your browser
- **Auto-Updates**: Always get the latest version automatically
- **Cross-Platform**: Works on Windows, Mac, Linux, Android, and iOS

## Installation Instructions

### Windows Desktop (Chrome, Edge, Brave)

1. Open the NRCS EAM website in your browser
2. Look for the install icon (⊕) in the address bar, or
3. Click the three-dot menu (⋮) → "Install NRCS EAM..."
4. Click "Install" in the popup dialog
5. The app will open in its own window
6. A shortcut will be added to your Start Menu and Desktop

**Alternative Method:**
- When you visit the site, a popup banner will appear at the bottom-right
- Click "Install" on the banner

### Android Mobile (Chrome, Samsung Internet)

1. Open the NRCS EAM website in Chrome
2. Tap the three-dot menu (⋮) in the top-right
3. Select "Add to Home screen" or "Install app"
4. Enter a name (default: "NRCS EAM")
5. Tap "Add" or "Install"
6. The app icon will appear on your home screen

**Alternative Method:**
- A popup banner will appear at the bottom of the screen
- Tap "Install" on the banner

### iOS/iPadOS (Safari)

**Note:** iOS requires Safari browser for PWA installation.

1. Open the NRCS EAM website in Safari
2. Tap the Share button (□↑) at the bottom of the screen
3. Scroll down and tap "Add to Home Screen"
4. Edit the name if desired (default: "NRCS EAM")
5. Tap "Add" in the top-right
6. The app icon will appear on your home screen

### Mac Desktop (Chrome, Safari, Edge)

**Chrome/Edge:**
1. Open the NRCS EAM website
2. Click the install icon (⊕) in the address bar, or
3. Click the three-dot menu → "Install NRCS EAM..."
4. Click "Install"
5. The app will open in its own window
6. Access from Applications folder or Launchpad

**Safari:**
1. Open the NRCS EAM website
2. Click "File" → "Add to Dock"
3. The app will be added to your Dock

### Linux Desktop (Chrome, Firefox, Edge)

1. Open the NRCS EAM website in your browser
2. Look for the install icon in the address bar, or
3. Click the browser menu → "Install NRCS EAM..."
4. Click "Install"
5. The app will open in its own window
6. Access from your applications menu

## Verifying Installation

After installation, you should see:
- **Desktop**: A standalone window without browser UI (no address bar)
- **Mobile**: An app icon on your home screen
- **Offline Mode**: The app loads even without internet (read-only)
- **NRCS Branding**: Nigerian Red Cross logo and navy blue theme

## Features in Installed Mode

### Online Mode
- Full access to all features
- Create and edit assets, work orders, maintenance schedules
- Real-time data synchronization
- Notifications for important events

### Offline Mode (Read-Only)
- View dashboard and statistics
- Browse assets, work orders, and maintenance schedules
- View asset details and QR codes
- Access inventory and vendor information
- View compliance records and financial data
- **Note:** Creating or editing requires internet connection

## Uninstalling the App

### Windows
1. Right-click the app icon → "Uninstall"
2. Or: Settings → Apps → NRCS EAM → Uninstall

### Android
1. Long-press the app icon
2. Tap "Uninstall" or drag to "Uninstall"

### iOS
1. Long-press the app icon
2. Tap "Remove App" → "Delete App"

### Mac
1. Drag the app from Applications to Trash
2. Or: Right-click → "Move to Trash"

### Linux
1. Right-click the app icon → "Uninstall"
2. Or remove from your applications menu

## Troubleshooting

### Install Button Not Showing
- **Chrome/Edge**: Ensure you're using HTTPS (secure connection)
- **Safari iOS**: Must use Safari browser (not Chrome or Firefox)
- **Already Installed**: Check if the app is already installed
- **Browser Support**: Update to the latest browser version

### App Not Working Offline
- Visit the app while online first to cache resources
- Check that service worker is registered (browser dev tools)
- Clear browser cache and reinstall if issues persist

### Updates Not Appearing
- Close and reopen the app
- Clear app cache: Settings → Privacy → Clear browsing data
- Uninstall and reinstall the app

### Notifications Not Working
- Check notification permissions in browser/device settings
- Ensure notifications are enabled in app settings (user menu → Notification Settings)
