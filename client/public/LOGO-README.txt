NRCS logo (50px on all auth pages)
----------------------------------
Place the official Nigerian Red Cross Society logo in this folder as:

  nrcs-logo.png

Use the circular red-cross logo (red cross, "NIGERIAN RED CROSS" text on white).
It will appear at 50px on:
  - Login
  - Signup
  - Forgot Password
  - Reset Password
  - Auth Callback

If the file is missing or fails to load, a fallback "NRCS" circle is shown.

Optional: set VITE_AUTH_LOGO_URL in .env to use a different image URL instead.
